DealHaven Extension Deployment

This is a standalone deployment page that doesn't rely on the React router. Use these tools to fix your manifest and prepare your extension for deployment.

1. Generate Icons

First, you need proper icons for your extension:

Open Icon Generator

Make sure to download all generated icons and place them in your public directory.

2. Fix Manifest and Critical Files

Run the manifest fixer script to create or fix manifest.json and ensure all critical files exist:

node public/fix-manifest.js

This script will:

  • Verify or create manifest.json
  • Generate missing icon placeholders
  • Create missing content and background scripts
  • Copy all critical files to the dist directory

3. Build the Extension

After fixing the manifest and generating icons, build your extension:

npm run build

The build process will compile your React application and copy all necessary files to the dist folder.

4. Load Unpacked Extension

In Chrome, follow these steps:

  1. Go to chrome://extensions/
  2. Enable "Developer mode" in the top right corner
  3. Click "Load unpacked" and select your dist directory
  4. Verify that the extension loads successfully

If you encounter "Manifest file is missing or unreadable", check that manifest.json exists in your dist folder and that all required icon files are present.

Troubleshooting

If you're still having issues after following these steps:

  1. Check Chrome's console for specific error messages
  2. Verify that all files referenced in manifest.json exist in your dist folder
  3. Try running the extension diagnostics script in your browser console
  4. Make sure your manifest.json is valid JSON
